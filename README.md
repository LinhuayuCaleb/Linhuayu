# 🎮 DodgeGame

一个基于 **Qt + C++** 开发的 2D 动作小游戏。
玩家需要在障碍物地图中移动、自动攻击敌人，并最终消灭所有敌人获得胜利。

---

## ✨ 游戏特色

* 🎯 **自动瞄准系统**

  * 玩家自动锁定最近敌人进行射击

* 🤖 **敌人 AI（带寻路）**

  * 敌人使用简单路径规划绕开障碍物追击玩家

* 💥 **子弹系统**

  * 玩家与敌人均可发射子弹
  * 子弹不会穿透障碍物

* 🧱 **碰撞系统**

  * 玩家 / 敌人 / 子弹 都无法穿过石块

* ❤️ **血量系统**

  * 玩家有血条显示
  * 被敌人攻击会掉血

* 🏆 **胜利机制**

  * 消灭全部敌人即可胜利

---

## 🎮 操作方式

| 按键 | 功能 |
| -- | -- |
| W  | 上移 |
| S  | 下移 |
| A  | 左移 |
| D  | 右移 |

👉 玩家会自动攻击，无需鼠标操作

## 🚀 编译运行

### 1️⃣ 克隆项目（或直接下载）

```
git clone https://github.com/你的用户名/DodgeGame.git
cd DodgeGame
```

---

### 2️⃣ 构建项目

```
mkdir build
cd build
cmake ..
cmake --build .
```

---

### 3️⃣ 运行游戏

```
./DodgeGame
```

（Windows 双击 exe 也可以）

---

## 📌 注意事项

* 不要上传 `build/` 目录
* 不要上传 `.exe` 文件
* 只保留源码文件即可
